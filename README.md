## 对象与目标

教程对象：有志于参加**信息学竞赛**的中小学生。

通过本教程，同学们可以了解如何从零开始进行信息学竞赛

- C++语言基础
- 竞赛入门算法
- 竞赛提高算法

## 教程大纲

本教程预计分成3个部分，以CCF发布的竞赛大纲为基础，进行知识点讲解，并附带练习与解答。

1. C++语言基础
2. 入门级算法
3. 提高级算法
